# Worthy Life

매일의 단어, 문장, 영어 뉴스, 미드 표현을 학습하고 날짜별 완료 이력을 관리하는 개인 영어 학습 사이트입니다.

## 실행

가장 쉬운 방법은 프로젝트 폴더의 `Worthy Life 실행.cmd`를 더블클릭하는 것입니다.

터미널에서는 다음과 같이 실행할 수 있습니다.

```powershell
cd D:\프로젝트\worthy-life
node scripts\dev-server.mjs
```

브라우저에서 `http://127.0.0.1:4173`으로 접속합니다.

학습 체크 이력과 저장한 단어는 브라우저의 `localStorage`에 자동 저장됩니다.
